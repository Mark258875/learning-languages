# Chinese Pinyin System (拼音)

## What is Pinyin?

Pinyin is the official romanization system for Mandarin Chinese. It uses Latin letters to represent Chinese sounds. **Master Pinyin before learning characters** — it's the phonetic foundation for everything.

---

## The Four Tones (声调)

Mandarin has 4 tones + a neutral (5th) tone. **Tone changes meaning completely!**

| Tone | Mark | Description | Contour | Example |
|------|------|-------------|---------|---------|
| 1st | ā | High and level | ——— | mā (妈 mom) |
| 2nd | á | Rising | ↗ | má (麻 hemp/numb) |
| 3rd | ǎ | Dips then rises | ↘↗ | mǎ (马 horse) |
| 4th | à | Falling sharply | ↘ | mà (骂 to scold) |
| Neutral | a | Short, unstressed | · | ma (吗 question particle) |

> ⚠️ Tones are not optional! Saying the wrong tone = different word entirely.

### Mnemonic for Tones:
1. **First**: flat and high — like a robot voice
2. **Second**: like a question "what?" going up
3. **Third**: like a sad "hmm?" dipping down then up
4. **Fourth**: firm and falling — like a command

## Initials (声母) — 21 Consonants

### Standard Initials
| Pinyin | Sound | Example |
|--------|-------|---------|
| b | "b" (unaspirated) | bā (八 eight) |
| p | "p" (aspirated puff) | pá (爬 to climb) |
| m | "m" | mā (妈 mom) |
| f | "f" | fàn (饭 rice/meal) |
| d | "d" (unaspirated) | dà (大 big) |
| t | "t" (aspirated) | tā (他 he) |
| n | "n" | nǐ (你 you) |
| l | "l" | lái (来 to come) |
| g | "g" (unaspirated) | gǒu (狗 dog) |
| k | "k" (aspirated) | kāi (开 to open) |
| h | "h" (slightly guttural) | hǎo (好 good) |
| j | "j" + "y" glide (like "jeep") | jī (鸡 chicken) |
| q | "ch" + "y" glide (aspirated) | qù (去 to go) |
| x | "sh" + "y" glide (like "she") | xǐ (洗 to wash) |
| zh | "j" (retroflex, curl tongue back) | zhōng (中 middle) |
| ch | "ch" (retroflex, aspirated) | chī (吃 to eat) |
| sh | "sh" (retroflex) | shū (书 book) |
| r | between "r" and "zh" (retroflex) | rén (人 person) |
| z | "dz" | zài (在 at/in) |
| c | "ts" (aspirated) | cǎo (草 grass) |
| s | "s" | shuǐ (水 water) |

### Key Distinctions
- **Aspirated vs unaspirated**: b/p, d/t, g/k, j/q, zh/ch, z/c — put hand in front of mouth: aspirated = puff of air
- **Retroflex (zh/ch/sh/r)**: curl tongue tip backward toward roof of mouth
- **j/q/x**: tongue flat, tip behind lower teeth, front of tongue raised

## Special Pinyin Rules

1. When **i** starts a syllable alone: write as **yi** (yī 一 one)
2. When **u** starts a syllable alone: write as **wu** (wǔ 五 five)
3. When **ü** starts a syllable: write as **yu** (yǔ 雨 rain)
4. After **j, q, x**: ü is written as **u** (the dots are dropped)
5. Tone mark goes on the main vowel of the final (a/e > o > i/u/ü)
