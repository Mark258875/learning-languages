# Russian Cases (Падежи)

Russian has **6 cases**. Cases indicate the grammatical role of a noun in a sentence, expressed through noun endings rather than word order.

Good news for Slovak speakers: Slovak also has 7 cases — you already understand the concept!

---

## The 6 Cases

| # | Case | Russian | Question | Role |
|---|------|---------|----------|------|
| 1 | Nominative | Именительный | Кто? Что? (Who? What?) | Subject |
| 2 | Accusative | Винительный | Кого? Что? (Whom? What?) | Direct object |
| 3 | Genitive | Родительный | Кого? Чего? (Of whom? Of what?) | Possession, absence |
| 4 | Dative | Дательный | Кому? Чему? (To whom? To what?) | Indirect object |
| 5 | Instrumental | Творительный | Кем? Чем? (By whom? By what?) | Means, accompaniment |
| 6 | Prepositional | Предложный | О ком? О чём? (About whom? About what?) | Location, topic |

## Case Endings — Singular

### Masculine Nouns (e.g. стол — table, брат — brother)

| Case | Hard (-consonant) | Soft (-ь) |
|------|-------------------|-----------|
| Nominative | стол | брат |
| Accusative (inanimate) | стол | брата |
| Accusative (animate) | брата | брата |
| Genitive | стола | брата |
| Dative | столу | брату |
| Instrumental | столом | братом |
| Prepositional | о столе | о брате |

### Feminine Nouns (e.g. книга — book, мама — mom)

| Case | -а ending | -я ending |
|------|-----------|-----------|
| Nominative | книга | мама |
| Accusative | книгу | маму |
| Genitive | книги | мамы |
| Dative | книге | маме |
| Instrumental | книгой | мамой |
| Prepositional | о книге | о маме |

### Neuter Nouns (e.g. окно — window, море — sea)

| Case | -о ending | -е ending |
|------|-----------|-----------|
| Nominative | окно | море |
| Accusative | окно | море |
| Genitive | окна | моря |
| Dative | окну | морю |
| Instrumental | окном | морем |
| Prepositional | об окне | о море |

## Key Uses of Each Case

### 1. Nominative — Subject
*Мама читает.* — Mom reads.
*Кот спит.* — The cat sleeps.

### 2. Accusative — Direct Object
*Я вижу кота.* — I see the cat.
*Он любит книгу.* — He loves the book.

> Animate masculine nouns in accusative = genitive form!

### 3. Genitive — Possession / Absence / Numbers
*книга мамы* — mom's book
*У меня нет кота.* — I don't have a cat. (lit. "By me there is no cat")
*пять столов* — five tables (genitive plural after 5+)

### 4. Dative — Indirect Object / Age
*Я дам книгу маме.* — I will give the book to mom.
*Мне двадцать лет.* — I am twenty years old. (lit. "To me twenty years")

### 5. Instrumental — By/With/Using / After "with"
*Я пишу ручкой.* — I write with a pen.
*Я иду с мамой.* — I go with mom.
*Он работает учителем.* — He works as a teacher.

### 6. Prepositional — Location / About
*Я живу в Москве.* — I live in Moscow.
*Он думает о маме.* — He thinks about mom.

## Common Prepositions by Case

| Preposition | Case | Meaning |
|-------------|------|---------|
| в, на | Accusative | into, onto (direction) |
| в, на | Prepositional | in, on (location) |
| из, с, от, без, для, до, около, после | Genitive | from, off, from, without, for, until, near, after |
| к, по | Dative | towards, along/by |
| с, за, над, под, перед, между | Instrumental | with, behind, above, under, before, between |
| о/об | Prepositional | about |

## Slovak — Russian Case Comparison

| Russian Case | Slovak Equivalent | Notes |
|---|---|---|
| Nominative | Nominatív | Same function |
| Accusative | Akuzatív | Same function |
| Genitive | Genitív | Same function |
| Dative | Datív | Same function |
| Instrumental | Inštrumentál | Same function |
| Prepositional | Lokál | Russian doesn't have Locative alone — it's always used with prepositions |
| — | Vokatív | Russian lost the vocative (mostly) |
